# Constantes

***Conteúdo da Aula:***

Constante é um valor fixo que não se modifica ao longo da execução do programa.

Você pode imaginar uma constante como uma variável que **“não varia”**:

* Uma constante só pode receber uma atribuição de valor, valor este que não poderá ser modificado.
* Da mesma maneira que as variáveis, as constantes têm nome e tipo.
